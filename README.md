## Installation

1. Clone this repo to `<space-todos>`

  `git clone https://github.com/meteor-space/todos.git <space-todos>`

2. Change into cloned directory

  `cd <space-todos>`

3. Install MGP (on time only)

  `npm install -g mgp`
 
4. Run MGP

  `mgp`
  
5. Run the app

  `./run.sh`
